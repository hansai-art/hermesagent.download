你在某個深夜裝好了 Hermes Agent,卡在一個錯誤訊息上,搜尋出來的中文資料只有零星幾篇過期文章——這個網站就是為了終結那個時刻而存在的。

## 這個站是什麼

HermesAgent.download 是 Hermes Agent 的**非官方中文社群知識庫**。所有內容都是 Markdown 檔,放在公開的 GitHub repo 裡,任何人都可以編輯、補充、修正。你現在看到的每一頁,右下角都有「編輯這一頁」的入口。

三件事讓它跟一般部落格不一樣:

1. **共筆**:內容不屬於任何個人。發現錯誤,改就對了。
2. **有品質關卡**:每個 PR 都會經過自動掃描(引用完整性、格式、空洞句偵測)與人工審稿,規範全部公開在 repo 的 `docs/editorial/`。
3. **自動進化**:每篇文章都標記適用版本與最後驗證日期。上游發布新版本時,系統會自動找出可能過期的文章,引導社群更新。

## 免責聲明

本站與 Nous Research **沒有任何關係**,不代表官方立場。所有下載連結一律導向官方來源;我們不重新散布任何安裝檔。
