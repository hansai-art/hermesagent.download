使用 openai-codex provider 搭配 gpt-5.4 時，即使串流 API 已透過 output_text.delta 事件送出文字，Hermes 仍會因為最終 response.output 清單是空的而丟出 RuntimeError，導致 3 次重試全部失敗，且串流文字被重複串接顯示成一段亂碼（如 hellohellohello）。此 issue 已關閉。
