在 hermes -z（oneshot 模式）下，即使 MCP server 通過 hermes mcp test 測試且能探索到工具，agent 實際可用的工具清單中卻沒有這些 MCP 工具。原因是 oneshot.py 在建立 AIAgent 時就已解析好工具清單，時間點在 MCP discovery 註冊動態工具之前，導致之後才加入的工具永遠不會被看到。
