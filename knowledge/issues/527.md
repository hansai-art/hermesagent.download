這個 issue 指出 Hermes Agent 目前的 gateway 授權是二元制（全部允許或完全阻擋），沒有權限層級概念。提案為 Telegram、Discord、WhatsApp、Slack 等 gateway 平台導入分層權限系統（Owner、Admin、User、Guest），讓部署者能與他人共用 agent 卻不必開放完整終端機存取等高風險能力。
