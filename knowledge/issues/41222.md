目前 Kanban 看板與 Desktop app 是分開的，使用者要另開終端機下指令（如 hermes kanban list）並手動複製任務 ID，造成多 agent 協作時的操作阻力。此 issue 提議在 Desktop app 加入側邊欄或 /kanban 指令，讓使用者不用離開聊天介面就能查看與操作看板任務。
