使用 Kimi 服務商的 kimi-for-coding 模型時，因為該模型強制要求 temperature 必須為 0.6，但 Hermes 目前沒有提供針對特定服務商或模型覆寫 temperature 的設定，導致所有 API 呼叫都收到 HTTP 400 錯誤。使用者建議新增可依服務商/模型調整 temperature 的設定選項。此 issue 已關閉，狀態為已修復。
