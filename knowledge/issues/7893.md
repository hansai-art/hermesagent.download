使用者在 ~/.hermes/.env 設定 GEMINI_API_KEY 使用內建 Gemini 服務商時，Google AI Studio 端點回傳 HTTP 400 錯誤；改用一般 Google Cloud API key（AIzaS 開頭）而非 Vertex AI 產生的金鑰（AQ.Ab8 開頭）才能正常運作。此 issue 已關閉，狀態為已修復。
