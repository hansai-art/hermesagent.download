這個 issue 指出 Codex gpt-5.5 壓縮閾值自動調高的說明通知，在 gateway session 因重建（如 gateway 重啟、cache 失效）時會重複顯示給使用者，即使並非開啟新的可見 session。提案讓此通知在同一個持久 gateway session 中只顯示一次。
