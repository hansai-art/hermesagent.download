回報者依官方文件設定 Teams 平台後，gateway 始終無法綁定 3978 埠，原因是 Teams 外掛匯入的 microsoft-teams-apps 套件需要 Python 3.12 以上，但 Hermes 安裝程式建立的虛擬環境是 Python 3.11，即使系統上已有 Python 3.13 也未被 Hermes 使用。
