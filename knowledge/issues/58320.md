當未設定任何 web backend API key 時，web_search 和 web_extract 會靜默失敗或回傳空結果，不會提示使用者搜尋功能不可用，可能導致使用者數週都不知道核心搜尋功能已失效。issue 建議在失敗時回傳明確錯誤訊息與解決步驟。
