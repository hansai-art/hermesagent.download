Slack 的 mrkdwn 格式不支援 GitHub Flavored Markdown 的 pipe 表格語法，導致 Hermes 在 Slack 回覆包含表格的內容（例如工具比較）時，表格會以純文字管線符號顯示而非正常表格。此 issue 已關閉。
