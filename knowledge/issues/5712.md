Hermes 的 cron 排程工作會在獨立的 session 中執行，執行結果送到 origin/home channel 時只有人類看得到，並附註「agent 無法看到這則訊息」，不會被寫進主要 gateway session 的對話紀錄，導致主 agent 常常不記得自己執行過排程任務，使用者得主動詢問才知道任務有沒有完成。

- 元件:gateway.優先級:P2
