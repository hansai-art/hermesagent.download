當 Hermes Desktop 連到遠端 Hermes 後端時，目前只能看到 profile 清單，實際生效的 profile 仍取決於遠端後端啟動時的 HERMES_HOME/HERMES_PROFILE，Desktop 的 Profiles 頁面無法真正切換後端使用的 profile。此 issue 提議讓 Desktop UI 能直接切換遠端使用中的 profile。
