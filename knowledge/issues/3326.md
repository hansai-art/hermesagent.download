使用 hermes chat -q 做程式化編排（如 CI pipeline、MCP server）時，目前只能得到純文字輸出，token 數、成本、session ID、模型資訊等都要自行從非結構化文字解析。此 issue 提議新增 --output-format json 旗標，輸出包含這些 metadata 的結構化 JSON，並已有對應 PR #2916。
