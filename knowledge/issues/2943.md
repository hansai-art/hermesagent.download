使用者回報排定的 cron job 無法將訊息送達指定的 Discord 頻道，不論是自動執行還是手動觸發都一樣；但透過在 Discord 上強制產生討論或手動測試訊息時卻可以正常送達。Gateway log 顯示 Discord API 回傳 404 錯誤。此 issue 已關閉，狀態為已修復。
