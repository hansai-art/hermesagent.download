這個 issue 指出 Hermes 的 display.show_reasoning 設定在 CLI 與訊息 gateway adapter 會正確顯示模型的推理過程，但 API server adapter（gateway/platforms/api_server.py）完全沒有實作這個邏輯，導致透過 OpenAI 相容介面（如 Open WebUI）連線的使用者永遠看不到推理內容。
