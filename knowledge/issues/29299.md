目前 Hermes 的 MCP OAuth 只支援 http://127.0.0.1:8765/callback 這種本機 HTTP callback，許多 OAuth 服務（如 Salesforce 官方 MCP 流程）不接受這種格式。issue 提議在 config 增加 redirect_uri 覆寫選項以支援 HTTPS。
