Hermes Desktop 的檔案系統掃描器會把偵測到的所有 git repo 廣播給每個 profile 各自的資料庫，導致切換 profile 時會看到其他 profile 的 repo；根因是 projects.record_repos 處理常式用 replace=True 覆寫了整份清單。
