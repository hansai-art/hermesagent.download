此提案指出 Hermes 目前的 session memory 是暫時性的，session 結束或 gateway 重啟後對話脈絡就會遺失，導致使用者必須不斷重新說明環境、偏好與專案狀態；提案建議建立一個以 markdown 檔案為單位、可搜尋且能自動壓縮的持久化「Vault」式 session memory 系統。
