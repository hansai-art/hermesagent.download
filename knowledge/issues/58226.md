/usage 儀表板在 Anthropic OAuth 帳號用量極低（0~1%）時，會誤顯示為 100% 已用、0% 剩餘。原因是程式碼把 API 回傳、本身已是百分比的數值誤判為 0-1 的小數並乘以 100，造成換算錯誤，回報者也指出 Codex 那條路徑並沒有這個問題。
