此 issue 回報：一開始用 Claude 訂閱驗證裝好 Hermes 後可以正常運作，但過一段時間後開始持續收到 HTTP 400：「You're out of extra usage. Add more at claude.ai/settings/usage and keep going.」，即使重啟 Hermes 並重新登入 Claude 驗證也無法恢復。作者強調這不是一開始就壞掉，而是使用一段時間後才開始出現，且同一帳號的 Claude Desktop 仍可正常使用，切換 Hermes 內的模型也沒有幫助，懷疑 Hermes 的 Anthropic 訂閱 / OAuth 路徑可能碰到跟 Claude Desktop 不同的用量閘門，或是 Hermes 沒有清楚處理 / 回報這個狀態。此 issue 已關閉。
