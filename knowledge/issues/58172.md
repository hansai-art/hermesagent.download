每次執行 hermes update 都會新增 git object 但從未清理舊物件，長期運行數月後 .git 目錄膨脹到 884MB（29GB 磁碟），加上 npm 快取與 session 檔案，磁碟使用率一度達 86%，讓後續升級變得有風險。
