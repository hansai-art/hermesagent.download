# TERMINOLOGY.md — 統一譯名表

> 中文技術翻譯站的生命線。內文出現左欄概念時,一律用「譯名」欄寫法。
> 新增條目走 PR;有爭議開 issue 討論。

## 原則

1. 官方已有中文名 → 從官方。
2. 社群慣用且無歧義 → 從社群。
3. 沒有共識的術語 → 保留英文原文,不硬翻。
4. 第一次出現時寫「譯名(English)」,之後直接用譯名。

## 禁用寫法

- 簡體中文技術譯名(「服务器」「宏」「缺省」)→ 用台灣慣用譯名(伺服器、巨集、預設)
- 「小白」→ 用「新手」
- 機翻腔(「让我们来看看」)→ 重寫成自然中文
