# 行為準則

本專案採用 [Contributor Covenant](https://www.contributor-covenant.org/zh-tw/version/2/1/code_of_conduct/) 2.1 版,以下為重點摘要;完整條文以上述連結為準。

## 我們的承諾

無論年齡、身體、身心障礙、族群、性別認同與表達、經驗程度、教育、社經地位、
國籍、外貌、種族、宗教或性傾向,每個人都能在這個社群無騷擾地參與。

## 我們的標準

**做**:

- 對事不對人;技術爭議以可查證來源為準
- 尊重不同的觀點與經驗程度——每個人都當過新手
- 優雅地接受建設性批評
- 犯錯就承認、道歉、修正,然後繼續

**不做**:

- 人身攻擊、貶低、嘲諷(包括嘲笑「這麼簡單都不會」)
- 騷擾(公開或私下)、未經同意公開他人資訊
- 性化的語言或圖像
- 其他在專業場合被合理認為不當的行為

## 執行

違規行為請回報維護者:開 issue(可只 @hansai-art 不寫細節,約私下管道)或
寄信給 repo 維護者。維護者有權移除、編輯、拒絕違反本準則的留言、commit、
issue 與 PR,情節重大者暫停或永久排除參與。

## 歸屬

本準則改編自 [Contributor Covenant](https://www.contributor-covenant.org) v2.1。
